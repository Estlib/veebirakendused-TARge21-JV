ERD diagramm CarsDB äpile:

Draw.io genereeritud või HTML tabelikujul

Andmebaas kasutab kahte tabelit.

Tabel 1 on tabel nimega "CarMark" ning sisaldab kahte võtit. Primary Keyd - milleks on integer id - ja "CarMarkName" keyd mis sisaldab kuni 50-tähelist stringi.

Andmed mis selles tabelis hoitakse, tähistavad eraldi autotootjaid, nagu Tesla, Toyota jne.

Tabel 2 on tabel nimega "CarInGarage" ning sisaldab endas kolme võtit. Primary key milleks on ID ning sisaldab integeri, foreign key "CarMarkId" mis sisaldab integeri, viidates eelmise tabeli - "CarMark"-i ID primary keyle ja "CarModelName" mis sisaldab endas kuni 50-tähelist stringi, kus hoitakse mudelinimetusi, nagu näiteks Yaris.
